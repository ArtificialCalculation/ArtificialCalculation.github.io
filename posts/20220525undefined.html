  • 汇报人名字：米晨晰

    1. 发现的问题

      • 实验想要证明的两个点
        1. 对于左右冠状动脉这种比较细小的血管分割，全心其他类的标注有助于它的分割（并且讨论哪些类有助于它的分割，哪些类有害于它的分割）
        2. 不同网络模型的feature map融合，再使用深度学习，可以提高左右冠状动脉分割的效果

    2. 解决的思路

      • 进行实验

    3. 完成的实验

      Details
      Baseline A: 2D Unet l 实验1：2类（左右冠状作为一类，其他标签与背景作为一类），五折交叉验证 (Dice loss, 保存feature map)
      Baseline B: 3D Unet l 均进行ROI+Seg的两阶段分割框架 l 实验1：2类（左右冠状作为一类，其他标签与背景作为一类），五折交叉验证 l 实验2：4类（某个其他标签，背景为一类），五折交叉验证 l 实验3：8类（各个标签为各自类，背景为一类），五折交叉验证
      Baseline C：3D Unet l 实验一：2类（左右冠状动脉为一类，其他标签与背景作为一类），五折交叉验证
      Proposed l Multi-task 2D +3D l 实验1：Baseline A.1 + Baseline B.3 feature map融合） l 实验2: Baseline A.1 + Baseline B.2 feature map融合

    4. 结果数据

      • 1-5标签的分割效果不错，平均dice在0.78-0.95
      • 6、7标签即左右冠状动脉标签分割效果只有0.6左右

    5. 接下的工作

      • 继续完善实验
